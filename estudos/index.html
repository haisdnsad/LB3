📘 Manual de PHP — Vetores e Laços de Repetição
🔹 1. Vetores (Arrays) em PHP

Em PHP, vetores são chamados de arrays e servem para armazenar vários valores em uma única variável.

📌 Tipos de Arrays
▶️ Array Indexado (índice numérico)
$frutas = ["maçã", "banana", "laranja"];

Ou:

$frutas[0] = "maçã";
$frutas[1] = "banana";
▶️ Array Associativo (chave => valor)
$pessoa = [
    "nome" => "João",
    "idade" => 20,
    "cidade" => "SP"
];
▶️ Array Multidimensional
$alunos = [
    ["nome" => "Ana", "nota" => 8],
    ["nome" => "Carlos", "nota" => 7]
];
🔹 2. Laços de Repetição em PHP

Servem para executar um bloco de código várias vezes.

🔁 while (enquanto)

Executa enquanto a condição for verdadeira.

$i = 0;

while ($i < 5) {
    echo $i;
    $i++;
}
🔁 do...while

Executa pelo menos uma vez.

$i = 0;

do {
    echo $i;
    $i++;
} while ($i < 5);
🔁 for (mais usado com arrays)

Ideal quando você sabe quantas repetições precisa.

for ($i = 0; $i < 5; $i++) {
    echo $i;
}
🔁 foreach (feito pra arrays 🔥)

Mais simples para percorrer arrays.

$frutas = ["maçã", "banana", "laranja"];

foreach ($frutas as $fruta) {
    echo $fruta;
}

Com chave:

$pessoa = ["nome" => "João", "idade" => 20];

foreach ($pessoa as $chave => $valor) {
    echo "$chave: $valor";
}
🔹 3. Interligação: Arrays + Laços

Aqui está o ponto mais importante 👇

👉 Arrays armazenam dados
👉 Laços percorrem esses dados

📌 Exemplo 1 — Percorrendo array com for
$frutas = ["maçã", "banana", "laranja"];

for ($i = 0; $i < count($frutas); $i++) {
    echo $frutas[$i];
}
📌 Exemplo 2 — Percorrendo com foreach (melhor opção)
$frutas = ["maçã", "banana", "laranja"];

foreach ($frutas as $fruta) {
    echo $fruta;
}

✔ Mais simples
✔ Mais legível
✔ Menos erro

📌 Exemplo 3 — Array associativo
$pessoa = [
    "nome" => "João",
    "idade" => 20
];

foreach ($pessoa as $chave => $valor) {
    echo "$chave: $valor 
";
}
📌 Exemplo 4 — Array multidimensional
$alunos = [
    ["nome" => "Ana", "nota" => 8],
    ["nome" => "Carlos", "nota" => 7]
];

foreach ($alunos as $aluno) {
    echo $aluno["nome"] . " - " . $aluno["nota"] . "
";
}
🔹 4. Resumo Final (pra decorar)

✔ Array = guarda vários valores
✔ for = quando você sabe o tamanho
✔ while = quando depende de condição
✔ foreach = melhor para arrays
✔ foreach + array = combinação mais usada em PHP

🧠 Dica importante (nível mais crítico)

Se você estiver usando:

for ($i = 0; $i < count($array); $i++)

👉 Isso pode ser menos eficiente, porque o count() roda toda vez.

Melhor:

$tamanho = count($array);

for ($i = 0; $i < $tamanho; $i++) {
    echo $array[$i];
}

Ou simplesmente:
👉 Use foreach (melhor prática)
